# # PhishGuard

A localized phishing simulation platform built for the MENA region.

## Overview

PhishGuard tests an organization's human security layer by sending realistic, localized phishing simulations in Arabic and French — impersonating institutions employees actually trust. It tracks employee responses in real time and generates professional, compliance-ready reports.

## Features

- Localized phishing email templates (Arabic & French)
- Automated campaign management and personalized sending
- Real-time tracking (opens, clicks, credential submissions)
- Fake landing pages with awareness redirects
- Live monitoring dashboard
- Automated PDF reporting with risk scoring

## Tech Stack

**Backend:** Python, Flask, SQLAlchemy
**Frontend:** AdminLTE, Chart.js
**Database:** SQLite (development) / PostgreSQL (production)
**Email:** smtplib, Mailhog (testing)
**Reporting:** WeasyPrint
